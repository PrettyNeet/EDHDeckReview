## Project Orientation

To understand this project and codebase:

1. Start with `docs/project-reference.md`.

2. Use `architecture.md` for deeper file-by-file and pipeline details.
3. Use `README.md` for setup, runtime, API usage, and product behavior details.

## Reading Order

Default order:

`docs/project-reference.md` -> `architecture.md` -> `README.md`
